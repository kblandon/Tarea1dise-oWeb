Todo lo que hacemos es de código abierto

Nuestra plataforma consiste totalmente de componentes de código abierto y se basa en una plataforma sólida de programas libres (como GNU/Linux). Además, colaboramos de manera activa con el ecosistema para mejorarlo para todos.

Seguro y respetuoso de la privacidad

Como el código fuente está disponible para su revisión, cualquiera (un investigador de seguridad, un usuario preocupado o un fabricante que incluye el sistema en sus equipos) puede verificar que el software es seguro y no filtra información personal.

Hecho para los desarrolladores

Tanto si su aplicación puede beneficiarse de una función o API nueva del sistema, como si tiene curiosidad por saber cómo se ha creado una prestación o una pauta de diseño existentes, usted tiene acceso completo a nuestro código fuente. Cópielo, aprenda de él, modifíquelo y redistribúyalo.

Respeto por la privacidad. En todos los ámbitos.

Sus datos siempre le pertenecen a usted, y solo a usted. No realizamos acuerdos publicitarios ni recopilamos datos personales confidenciales. Nos financian directamente nuestros usuarios que pagan lo que quieren por elementary OS y por las aplicaciones del Centro de Aplicaciones. Y así es como debe ser.
Nuestra normativa de privacidad ➔

Sin escuchas

elementary OS le a ayuda a controlar el comportamiento de las aplicaciones. Cuando una aplicación use el micrófono, se muestra un indicador para informar acerca de ello. Cuando una aplicación utilice demasiada energía, el indicador de energía lo marcará.

Permisos

Cuando una aplicación quiera acceder a sus datos o dispositivos, deberá pedir permiso de forma explícita. Revisamos todas las aplicaciones del Centro de Aplicaciones para garantizar que utilicen adecuadamente los permisos, y siempre podrá revocarlos en la Configuración del Sistema.

Mantenimiento

elementary OS mantiene ordenados los archivos temporales y los de la papelera de forma automática. Esto no solo ahorra espacio del dispositivo, sino que ayuda a garantizar que sus datos privados no vuelvan a atormentarlo después.

Fáciles de usar. Ideales para un uso eficiente del teclado.

elementary OS está diseñado para que sea fácil de entender y de usar para el nuevo usuario. Esto no quiere decir que sea limitado; los potentes y personalizables atajos de teclado garantizan mantenerse productivo, ya sea el primer o el enésimo día que usa elementary.
